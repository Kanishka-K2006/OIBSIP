1)Number Guessing Game (Java) :

This is a simple Number Guessing Game developed using Java, where the user tries to guess a randomly generated number within a limited number of attempts.

Features :

->Generates a random number between 1 and 100

->Allows the user 5 attempts to guess the correct number

->Provides hints if the guessed number is greater or lesser

->Displays remaining attempts after each guess

->Shows the correct number if all attempts are exhausted

Technologies Used :

->Java

->VS Code

->Git & GitHub
